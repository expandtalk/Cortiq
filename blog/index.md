# CortIQ Documentation Blog

**Blog Index & Configuration**

This folder contains SEO-optimized blog posts and documentation for CortIQ.

## Blog Posts Structure

Each blog post should follow this format:

```markdown
---
title: "Post Title (50-60 characters)"
description: "Meta description (150-160 characters)"
author: "Author Name"
date: "2025-02-09"
category: "Category Name"
tags: ["tag1", "tag2", "tag3"]
slug: "post-url-slug"
image: "path/to/image.jpg"
imageAlt: "Image alt text for SEO"
readingTime: "5 min read"
---

# Post Title

Post content here...
```

## SEO Guidelines

- **Title**: 50-60 characters, include primary keyword
- **Description**: 150-160 characters, natural language
- **Slug**: lowercase, hyphens, no special characters
- **Heading Structure**: H1 (title) → H2 (sections) → H3 (subsections)
- **Keywords**: Primary (H1), Secondary (2-3 in body), Long-tail in subheadings
- **Internal Links**: Link to other blog posts and docs
- **Image Alt Text**: Descriptive, include keyword when natural

## Publishing Checklist

- [ ] Title is 50-60 characters
- [ ] Meta description is 150-160 characters
- [ ] Image alt text is descriptive
- [ ] At least 3 internal links
- [ ] Headings follow H1 → H2 → H3 structure
- [ ] Slug is URL-friendly
- [ ] Category and tags are assigned
- [ ] Reading time is accurate
